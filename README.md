# Beijing Cut Flowers Monitoring Project

This repository contains the code and supporting files for the Beijing Cut Flowers Monitoring Project.

The project is a flower monitoring system that captures images, analyzes flower hydration status, and records results for review in a web application. It is designed to help users track flower health and receive alerts when problems are detected.

## Getting Started

This repository is intended to provide the core project files only. For deployment instructions, setup steps, and full technical documentation, please refer to the project wiki.
